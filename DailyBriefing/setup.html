  2. 카카오 로그인 활성화 → 동의항목에서 카카오톡 메시지 전송 추가
  3. 카카오 API 도구에서 액세스 토큰 발급
  4. 발급된 토큰은 12시간마다 갱신 필요 (make.com에서 자동 갱신 설정 가능)

테스트 실행
  1. 시나리오 오른쪽 상단 Run once 버튼 클릭
  2. 각 모듈의 실행 결과 확인 (초록 체크 = 성공, 빨간 X = 오류)
  3. Claude 응답이 JSON인지 확인: 모듈 클릭 → Output에서 market, news, stocks, checklist 확인
  4. Notion, Slack, 카카오톡에 메시지가 도착했는지 확인
💡 테스트가 성공하면 시나리오 하단 Scheduling에서 스케줄을 활성화하고 Save하세요.
